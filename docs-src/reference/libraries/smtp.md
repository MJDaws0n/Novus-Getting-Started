# smtp

**Purpose:** lightweight SMTP email sending helpers.

**Typical import:**

```novus
import lib/smtp smtp;
```

## Sending mail

- `smtp_send_email`
- `smtp_send_line`

## SMTP protocol helpers

- `smtp_read_response`
- `smtp_response_code`

## Utility helpers

- `base64_encode`
- `parse_ip_octet`
