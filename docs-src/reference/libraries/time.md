# time

**Purpose:** timing, elapsed-time measurement, and sleep helpers.

**Typical import:**

```novus
import lib/time time;
```

## Clock access

- `get_time_ms`
- `get_time_s`
- `get_time_us`

## Elapsed-time helpers

- `elapsed_ms`
- `elapsed_us`
- `timer_start`
- `timer_elapsed_ms`
- `timer_elapsed_s`
- `timer_elapsed_us`

## Sleep and formatting

- `sleep_ms`
- `sleep_s`
- `format_duration_ms`
