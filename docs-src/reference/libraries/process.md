# process

**Purpose:** subprocess launching, output capture, waiting, and a few desktop picker helpers.

**Typical import:**

```novus
import lib/process/main process;
```

## Running commands

- `run_cmd`
- `shell_exec`
- `shell_output`

## Capturing output

- `capture_output`
- `capture_output_full`

## Waiting

- `wait_pid`
- `waitpid_nohang`

## UI picker helpers

- `pick_file_dialog`
- `pick_folder_dialog`
