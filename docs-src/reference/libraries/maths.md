# maths

**Purpose:** small arithmetic and number helper library.

**Typical import:**

```novus
import lib/maths maths;
```

## Absolute value and sign

- `abs`
- `abs64`
- `sign`
- `sign64`

## Min / max / clamp

- `min`
- `min64`
- `max`
- `max64`
- `clamp`
- `clamp64`
- `wrap`

## Integer utility functions

- `gcd`
- `lcm`
- `factorial`
- `fibonacci`
- `is_prime`
- `is_even`
- `is_odd`
- `digit_count`
- `digit_sum`
- `div_ceil`
- `isqrt`

## Mapping and interpolation

- `lerp`
- `map_range`

## Powers

- `pow_i32`
- `pow_i64`
