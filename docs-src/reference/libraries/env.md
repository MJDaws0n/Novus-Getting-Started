# env

**Purpose:** OS and process environment helpers.

**Typical import:**

```novus
import lib/env env;
```

## Identity and environment

- `get_username`
- `get_home_dir`
- `getenv`
- `gethostname`
- `getcwd`

## Process and user IDs

- `getpid`
- `getppid`
- `getuid`
- `geteuid`

## Platform information

- `get_arch`
- `get_os_name`
- `get_os_version`
